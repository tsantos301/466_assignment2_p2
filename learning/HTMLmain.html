Introduction:

This is a primer in a series of tutorials that will calmly introduce you to the very basics of HyperText Mark-up Language. Eventually, you'll easily know enough to create your own HTML web page.

The following are the main sections and ideas for this tutorial:

  • Hello World - easing into your first page.
  • Document Types - Learning the foundations of a web page
  • Comments and Tags - Making readable code, and understanding tags
  • Links - intro to navigating the web
  • Paragraphs and lists - intro to some page essentials
  • Tables - getting a little bit more technical
  • Forms - a common item used in many sites
